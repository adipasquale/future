Description du projet

Une salle de bain se déplace vers une fenêtre, libère une lucarne pour que la cuisine étendue s'ouvre sur le salon. Le poteau nécessaire pour supporter la retombée de cloison qui les sépare se camoufle dans un montant de bibliothèque. Ses rayonnages peuvent ainsi être ouverts en partie, pour souligner sans aveugler cette séparation.
